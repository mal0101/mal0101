## 🧠 Projects That Excite Me

### 🧪 [Plagiarism Detection System](https://github.com/mal0101/plagiarism-detector)

> 🚧 **In Progress** — Building an end-to-end system that detects plagiarism using NLP + ML!

- 🧹 Text preprocessing: tokenization, stop-word removal, cleaning
- 🔍 Similarity metrics: Jaccard, cosine similarity, overlap coefficient, n-grams
- 🧠 ML modeling: scikit-learn, feature engineering, TF-IDF, Random Forest
- 🌐 Deployment-ready: Flask/Streamlit API + Web UI
- 🎯 Bonus: Testing with paraphrased texts, synonym detection, and model evaluation (F1-score > 0.8)
